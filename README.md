Para Alguém Especial 💐
Uma landing page interativa e romântica inspirada na tendência "Flower Code" do TikTok. Este projeto utiliza animações complexas em CSS para criar um buquê de flores neon que desabrocha ao toque do utilizador.

📋 Descrição
O projeto consiste numa experiência visual onde, inicialmente, a página apresenta um ambiente noturno calmo com uma instrução de interação. Ao clicar ou tocar no ecrã, as animações de crescimento são ativadas, revelando um buquê de flores detalhado e uma frase clássica da literatura romântica.

✨ Funcionalidades
Animação de Desabrochar: Flores, folhas e grama que crescem de forma dinâmica utilizando animações CSS keyframes.

Interatividade: O estado inicial da animação é pausado e apenas inicia após a interação do utilizador.

Citação Literária: Exibição de uma frase marcante da obra Orgulho e Preconceito, de Jane Austen, com efeitos de brilho neon.

Ambiente Visual: Fundo com gradiente radial simulando uma noite estrelada e partículas de luz que acompanham as flores.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica para os elementos da flor e containers de texto.

CSS3 / SCSS: Estilização avançada, transformações 3D e animações de cronometragem precisa.

JavaScript: Lógica simples para manipulação de classes do DOM e controlo do fluxo da animação.

🚀 Como Utilizar
Clona este repositório.

Certifica-te de que a estrutura de pastas mantém o ficheiro index.html na raiz e os estilos em css/main.css.

Abre o ficheiro index.html em qualquer navegador moderno.

Clica em qualquer lugar do ecrã para ver as flores crescerem.
